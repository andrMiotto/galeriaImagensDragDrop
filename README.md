# Galeria de imagens drag and drop

Este repositório contém uma galeria de imagens com funcionalidade de drag & drop (arrastar e soltar). Com esta aplicação, você pode organizar imagens de maneira intuitiva, simplesmente arrastando-as entre diferentes áreas da interface.
https://galeria-imagens-drag-drop.vercel.app/

## Funcionalidades

- Exibição de uma galeria de imagens.
- Reorganização das imagens por drag & drop.
- Interface intuitiva e responsiva.
- Suporte a múltiplos navegadores modernos.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (puro)
- Outras bibliotecas podem ser incluídas conforme necessidade.

## Como rodar o projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/andrMiotto/galeriaImagensDragDrop.git
   ```
2. Acesse o diretório do projeto:
   ```bash
   cd galeriaImagensDragDrop
   ```
3. Abra o arquivo `index.html` no seu navegador preferido.

## Estrutura de pastas

- `/assets` - Imagens e outros arquivos estáticos usados na galeria.
- `/css` - Arquivos de estilo.
- `/js` - Lógica JavaScript para o drag & drop e outras funcionalidades.
- `index.html` - Página principal da aplicação.
